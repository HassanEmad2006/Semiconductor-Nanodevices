# Overview

This repository documents an undergraduate team project exploring the fundamental principles of semiconductor physics, semiconductor materials, and nanoscale device fabrication.

The project investigates how material properties influence electronic device performance, compares several semiconductor materials used in nanoelectronics, and reviews fabrication technologies employed in modern semiconductor manufacturing.

Rather than presenting original research, this repository serves as a structured technical literature review and educational reference developed during undergraduate studies in Nanotechnology and Nanoelectronics Engineering.

# Semiconductor Materials Overview

The project compares several semiconductor materials commonly used in modern electronic devices.

| Material | Typical Applications | Main Characteristics |
|-----------|---------------------|----------------------|
| Silicon (Si) | Integrated circuits, solar cells | Mature manufacturing, low cost |
| Gallium Arsenide (GaAs) | RF devices, LEDs, microwave electronics | High electron mobility |
| Graphene | Flexible electronics, sensors | Extremely high mobility, zero band gap |
| MoS₂ | Next-generation nanoelectronics | Atomic-scale thickness with finite band gap |

The report discusses the advantages, limitations, and engineering trade-offs associated with each material.

# Fabrication Technologies

The repository reviews the fabrication sequence of modern semiconductor devices using FinFET technology as a representative example.

Major fabrication topics include:

- Silicon wafer preparation
- Fin formation
- High-k dielectric deposition
- Metal gate formation
- Source/Drain implantation
- Spacer formation
- Interconnect fabrication
- Deep Ultraviolet (DUV) Lithography
- Extreme Ultraviolet (EUV) Lithography
